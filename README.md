# Aurared VSCode Theme

My beautiful custom theme . Feel free to use :)  

This theme can be use without publish using guide below.

## Tools  
- nodejs
- vsce 

## Guide  
vsce package  
```powershell
npm install -g @vscode/vsce
```  
configure Theme 
```powershell
git clone https://<username>:<token>@github.com/almalikzakwan/Aurared.git
cd Aurared
npm install
vsce package
code --install-extension .\amz-{version}.vsix
```  

or you can download and install extension file (.vsix) from releases
```powershell
code --install-extension .\amz-{version}.vsix
```
